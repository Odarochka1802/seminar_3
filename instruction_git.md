# **Инструкция по работе с системой контроля версий Git**

Git — это набор консольных утилит, которые отслеживают и фиксируют изменения в файлах. С его помощью мы можем сравнивать, анализировать, редактировать, сливать изменения и возвращаться назад к последнему сохранению. 
 
## Инициализация репозитория

Чтобы инициализировать новый репозиторий нужно ввести в терминал команду:

    git init

## Проверка состояние репозитория

Чтобы проверить текущее состояние репозитория используем команду:

    git status

### Результат:


    $ git status
    # On branch master
    nothing to commit (working directory clean)

Команда проверки состояния сообщит, что коммитить нечего. Это означает, что в репозитории хранится текущее состояние рабочего каталога, и нет никаких изменений, ожидающих записи.
