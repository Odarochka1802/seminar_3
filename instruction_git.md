# **Инструкция по работе с системой контроля версий Git**

Git — это набор консольных утилит, которые отслеживают и фиксируют изменения в файлах. С его помощью мы можем сравнивать, анализировать, редактировать, сливать изменения и возвращаться назад к последнему сохранению. 
 
## Инициализация репозитория

Чтобы инициализировать новый репозиторий нужно ввести в терминал команду:

    git init
